# 🎓 ScholarAssist

A full-stack web application that helps students discover scholarships from multiple sources in a centralized platform.

---

## 🚀 Overview

ScholarAssist simplifies the process of finding scholarships by collecting and displaying relevant opportunities in one place.

Instead of manually searching across different websites, users can browse structured scholarship data and find opportunities more efficiently.

---

## ✨ Features

* 🔍 **Scholarship Listing**

  * Displays scholarships in a structured and user-friendly format

* 🌐 **Web Scraping**

  * Fetches scholarship data from external websites using JSoup

* 🔄 **Scheduled Updates**

  * Keeps scholarship data updated using backend scheduler

* 📋 **User-Friendly Interface**

  * Clean and responsive UI built with React

---

## 🏗️ Tech Stack

### Frontend

* React.js
* HTML, CSS, JavaScript

### Backend

* Spring Boot (Java)
* REST APIs

### Database

* PostgreSQL

### Tools & Technologies

* JSoup (Web Scraping)

---

## 🧠 System Design Highlights

* Backend built using **Spring Boot REST architecture**
* Scraper service extracts and stores scholarship data
* Scheduler automates periodic data updates
* Separation of frontend and backend for scalability

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ScholarAssist.git
cd ScholarAssist
```

### 2️⃣ Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```
